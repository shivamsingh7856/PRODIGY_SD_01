# Temperature Converter - Java Project

## Description
The Temperature Converter is a Java-based application that converts temperature values between Celsius, Fahrenheit, and Kelvin. The program allows the user to enter a temperature value and select the original unit of measurement. It then converts the temperature into the other two units and displays the converted values.

This project is designed to demonstrate basic Java programming concepts such as user input, conditional statements, mathematical calculations, and object-oriented programming. The program is user-driven, meaning the user can interact with the application by entering values and selecting units.

## Objective
The objective of this project is to build a temperature conversion tool that helps users easily convert temperature values between different units.

## Features
- Accepts user input
- Supports Celsius, Fahrenheit, and Kelvin
- Converts temperature into all units
- Displays accurate results
- Simple and user-friendly interface

## Technologies Used
- Java

## Conclusion
This project helped in understanding GUI development, user interaction, and temperature conversion logic in Java.
